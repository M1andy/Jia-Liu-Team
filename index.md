# 贾棋/刘宇计算机视觉实验室

实验室主要研究目标分为通用人工智能和专用人工智能，
前者包括多模态大模型、几何特征学习和生成式模型，
后者包括医疗健康计算、无人机视觉感知和水下目标感知。
具体来说，实验室的主要研究领域包括图像匹配、图像分割、目标检测、图像增强、少/零样本学习、持续学习、半监督学习、图像生成、多模态学习及医学影像等方面。
